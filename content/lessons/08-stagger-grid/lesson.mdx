This is variants from lesson 03 scaled up: a parent grid that staggers its
children, plus nine cells that each pop in with a spring. Re-trigger the whole
cascade by toggling the parent's `animate` state.

<Instructions>

1. Open **Exercise.jsx** and import `motion` from `"motion/react"`.
2. `gridVariants`: `hidden` → `{ opacity: 0 }`, `show` → `{ opacity: 1, transition: { staggerChildren: 0.08 } }`.
3. `cellVariants`: `hidden` → `{ opacity: 0, scale: 0.3 }`, `show` → `{ opacity: 1, scale: 1, transition: { type: "spring" } }`.
4. Parent grid → `motion.div variants={gridVariants} initial="hidden" animate={active ? "show" : "hidden"}`.
5. Each cell → `motion.div variants={cellVariants}`.

</Instructions>

Press **Trigger Grid** to replay the cascade.
